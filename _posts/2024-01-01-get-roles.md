Retrieve a list of all available roles within your company. This endpoint provides role information including IDs, names, and role types, which is essential for user management and permission assignment.

## Request Example

```http
GET https://api-external.monitorqa.com/roles
X-API-KEY: abcdef12345
```

## Response

### Role Types
- `0` - Admin
- `1` - Auditor
- `2` - Auditee
- `3` - Observer

**Success Response**

```http
HTTP/1.1 200 OK
Content-Type: application/json

[
  {
    "id": "role-admin-001",
    "name": "System Administrator",
    "roleType": 0
  },
  {
    "id": "role-auditor-001",
    "name": "Senior Auditor",
    "roleType": 1
  },
  {
    "id": "role-auditee-001",
    "name": "Department Manager",
    "roleType": 2
  },
  {
    "id": "role-observer-001",
    "name": "Quality Observer",
    "roleType": 3
  }
]
```

For errors responses, see the [response status codes documentation](#/response-status-codes).
